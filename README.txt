How to run the code
	under the directory src\main\java\assignment4
	running EasyGridWorldLauncher.java in Intellij or any IDE you prefer for building the easy grid world
	running HardGridWorldLauncher.java in Intellij or any IDE you prefer for building the Hard grid world
How to generate graph
	In terminal of the IDE, get data and put them in analysis.py
	then run analysis.py by: python analysis.py
